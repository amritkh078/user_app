# Brief

API for user management

## Requirements

 - git
 - Nodejs


## Installation

#### Clone the repository

` git clone  https://github.com/amritkh078/user_app`

#### Change directory

`cd user_app`

#### Install the packages

`npm install`

#### Configure app

Open `.env.example` and rename it to `.env` and set the environment variables


#### Run the program

` npm start`
