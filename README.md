# Brief

API for record storage where user can create, update, delete and view records.

## Requirements

 - git
 - Nodejs


## Installation

` git clone  https://github.com/amritkh078/user_app` 

`cd user_app`

`npm install`

## Configure app

Open `.env.example` and rename it to `.env` and set the environment variables


## Run the program

` npm start`
